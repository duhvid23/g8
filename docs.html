G8 Docs

Rep‑gated, auto‑graduating launchpad on World Chain. Users mint on a bonding curve (in WLD). When the threshold hits, the same tx creates a Uniswap v3 pool, seeds liquidity, and burns the LP position.

Why World Chain

  • Verified users: World ID = real humans by default.
  • Wallets on day 1: Privy in‑app wallets mean normies onboard without friction.
  • World App Store: Users get instant access to apps. Perfect for new traders.

Launch Flow

  1. Verify: Pass World ID → Privy wallet created.
  2. Mint: Users buy with WLD on a bonding curve → tokens minted instantly.
  3. Graduate: Crossing threshold auto‑creates Uniswap v3 pool (WLD/TOKEN), seeds LP (85% WLD + 12–15% supply), burns LP NFT, pays rakes (10% platform, 5–7% creator vesting).
  4. Trade: Price continuity from curve. Pool is live; LP is permanently locked.

Fees & LP

  • Platform rake: 10% of raised WLD (immediate to treasury).
  • Creator rake: 5–7% of raised WLD (streamed/vested).
  • Liquidity: 85% of WLD + reserved token bucket → LP.
  • DEX: Uniswap v3, 0.3% tier, tight initial range around final price.

Security

  • LP position NFT is burned (or black‑holed) → no liquidity rugs.
  • Atomic pool init + add liquidity in the same tx → no snipeable gap.
  • Phase machine: Funding → Graduating → Live. Double‑grad protected.

FAQ

Do users get tokens immediately? Yes. Tokens mint to their Privy wallet on each buy.

Can tokens be moved pre‑graduation? Yes, but there’s no public pool until graduation.

Why burn LP? To guarantee a price floor and remove rug risk.
